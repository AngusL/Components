# Components
3D component models in STEP and WRL format for KiCad EDA.

| Transform | Axis |      Value |
| :-------- | :--: | ---------: |
| scale     |    X |   0.393701 |
|           |    Y |   0.393701 |
|           |    Z |   0.393701 |

|           |    Z |  90.       |
